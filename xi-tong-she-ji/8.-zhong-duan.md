{
    crit,  // create item
    cren,  // create enemy

    set,   // set value

### 展望

现在的终端只有栈式读取和解析指令，需求稍微多一些就难以忍受了。

目前有两种重构的想法：

1. 制作终端指令类，提供--help等一系列功能
2. 用Attribute绑定特定函数，在终端中通过反射寻找
